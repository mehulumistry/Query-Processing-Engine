How to run the program?

Follow the below steps:

STEP 1--> Run the ProcessingEngine.java file.
            a) It will ask for input file name or paste the query. Take care of the extra white lines.
            b) The file will produce "Output.java" file and "MFStructure.java".
            c) Sometimes you will have to reload or save the MFStructure.java file and Output.java when running Queries one by one.

STEP2--> Run Output.java file
            a) It will ask for output file name.
            b) Enter the name of the file where the result of the query is to be stored.
